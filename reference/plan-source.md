# Source Note

This project is a Vite React UI app.

- The app UI and behavior are in `src/`.
- Plan content is maintained in `src/data/planData.js`.
- Topic sections are componentized under `src/components/topics/`.
- Root `index.html` is only the Vite mount shell for deployment.

For Netlify deployment, use `npm run build` and publish the `dist/` folder.
